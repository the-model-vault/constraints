# Common constraints

## Why to use constraints
They prevent us from accidental using incompatible versions of python libraries.

## How to use constraints
```
pip install <something> --constraint https://raw.githubusercontent.com/the-model-vault/constraints/refs/heads/main/requirements.txt
```

## What to do with other Python / libraries versions?
We may create other constraint list in case we need to build a code using other versions of Python and/or libraries.
